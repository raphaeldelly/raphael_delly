css Les CSS (Cascading Style Sheets en anglais, ou « feuilles de style en cascade ») sont le code utilisé pour mettre en forme une page web.
web Le web désigne un système de pages reliées entre elles par des hyperliens (ou liens hypertextes, qui mènent vers des adresses web). En cliquant avec la souris sur un hyperlien figurant sur une page, le navigateur nous emmène ailleurs, soit sur le même site, soit sur un autre.
